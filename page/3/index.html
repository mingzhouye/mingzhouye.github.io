第4章 秋水仙在取景框里倒置着燃烧

发表于 2022-01-25 | 分类于 大正浪漫

    比起座敷我更喜欢歌舞弹唱，他在信里这样对目黑说。

    目黑不知道“座敷”是什么意思，于是去雅虎上搜。
    座敷的本意是指某种形式的客厅，对于艺伎而言这个词还意味着侍宴陪客。

    道枝把艺伎的工作分为两个部分，一部分是艺术，一部分是座敷。
    艺术是他喜欢的部分。一名普通的艺伎通常要从十岁开始，花费整整五年的时间完成诗书乐舞各种课程。
    道枝学得很快，从一个洒扫院子、端碗送碟的侍者到正式“换领”成为艺伎，他只用了不到一年的时间。

    座敷则是他不那么擅长的部分，他不够巧言善辩，不够诙谐幽默，很多时候只是像个精致的瓷娃娃一样安静地坐在一边。
    尽管如此还是不断地有客人要求道枝陪席，那些男人并不希望年轻的艺伎过分健谈，当他们吹嘘自己的权力与地位的时候，他只要温顺地微笑，在合适的时间斟上一杯酒就足够了。
    他跪坐在宴会厅的榻榻米上，喘息在男性的虚荣与欲望构成的漩涡中，觉得自己和一只昂贵的花瓶没有什么分别。
    那些男人在交际场上用很多东西把自己体面地包装起来，一只皮包、一件西装、一把烟斗、一个艺伎。比起一个有血有肉的人，他更像一个艺术化的符号。

    今天的客人是个发顶稀疏的银行家，他说道枝像从木刻版画里走出来的女孩子。
    在玄关穿好鞋子以后他问跪坐在一旁的侍者要了一个信封，道枝知道那是客人给他的小费，于是恭敬地又鞠了一躬。
    等到银行家的车子消失在视野尽头以后他才拆开祝仪袋，里面竟然装了10日元。

    道枝手忙脚乱地把钱塞回去，心里七上八下地回到房间里给目黑写信。

    他一定还有什么企图，所以预先给了我这么多的小费。
    那个客人一定以为，只要这笔钱在我这里，就意味着我接受了他的某种暗示了。
    和先斗町相邻的宫川町流行一个词语叫做“双证艺伎”，他们除了为客人表演歌舞之外还会做点别的。

    目黑说在他的时代，10日元连自动贩卖机里的一瓶水都买不来。
    他不知道这些钱在一百年前的大正能买些什么，但是他查到了日俄战争时期日本军人的基本月薪，陆军新兵和海军新兵每个月的薪水分别是1.2日元和2.1日元。
    这样看来10日元确实是相当一笔巨款了，他的心也跟着提起来。

    好在没过多久道枝就又写信过来和他报平安了。银行家有太多的钱供他挥霍，他完全没有别的企图，只是通过这种方式满足自己的自尊心而已。
    梅川告诉他，自他鞠躬从那个客人手中接过祝仪袋开始，事情就已经全部结束了。
    目黑不知道为什么，心里也跟着松了一口气。

 

    道枝想了很久要用这10日元买些什么。
    他沿着先斗町的窄街一路走到尽头，离开艺伎区，离开鸭川河。
    气势恢宏的赤炼瓦建筑巨兽一样俯瞰着往来的人群，可乐饼的香味钻进他的鼻子。这种舶来的油炸食品和咖喱饭、炸猪排并称为“大正三大洋食”，他还从来没有吃过。
    硬币在他的口袋里哗啦作响，道枝吞了吞口水，头也不回地转身上了电车。
    移动的铁皮盒子带着他摇摇晃晃了十几分钟，他在一家照相馆门前下车，鼓起勇气推开了店门。

    目黑在信里感叹过，真想看看道枝君穿和服是什么样子啊。

    “您好？”
    摄影师是个穿西装马甲、留小胡子的微胖男人，道枝数了5日元给他。
    “就穿这个拍吗？”
    道枝拘谨地点了点头。
    他穿了自己最喜欢的一件和服，水蓝色的西阵织布料上开着大束大束的秋水仙，领口露出一截雪白的脖子，纤细柔韧的线条像水仙花的花茎。

    摄影师把他领到一块从天花板垂到地面的布幔前，摸着那一络小胡子将他从头打量到脚。
    水蓝色很适合他，颜色清淡的和服让他看起来有种特别的气质，瓷一样的易碎，水一样的洁净。
    他犹豫了一下，从杂物箱翻出一块半隐半透的、红色的纱。
    “把它披在身上。”

    道枝照着做了。
    摄影师绕到箱式相机的后面，把脑袋钻到黑布里蒙住。
    曝光用的镁粉剧烈地闪了一下，咔嚓一声过后闪光灯的灯罩开始冒烟。

    他看到一枝含着露水的秋水仙在取景框里倒置着燃烧。

第3章 干涸的金鱼架在灯上烤

发表于 2022-01-25 | 分类于 大正浪漫

    他们的关系从“敬启”进展到“前略”，再到完全省略掉那些繁复冗杂的格式和问候。

    所以说，目黑君目前在一家名叫“杰尼斯”的茶屋工作？这名字真奇怪。
    那之前提到的康二君是目黑君的“姐妹”，泷泽君是目黑君的“妈妈”？

    目黑仰头灌了半瓶水，腮帮子鼓鼓的还没有来得及咽下去就着急地拆开信封，接着剧烈地咳嗽起来。
    他坐在书桌前提笔回信，很快意识到道子把“姐妹”和“妈妈”的称谓套用在康二和泷泽身上并不是在和他开玩笑。

    姐妹是艺伎花街最基本的结构单位，这个行业用家庭成员的概念代替前辈与后辈、师傅与学徒之间的相对关系。
    事实上这种模糊了亲属关系的现象在日本大部分传统职业中普遍存在，虚拟的“姐妹”和“妈妈”构成了道子所处在的微缩社会。
    他在尝试着用自己能够理解的方式，探索那个属于目黑的、于他而言完全陌生的世界。

    泷泽先生是副社长。
    社长原本是喜多川先生，喜多川先生去世后由他的家人继承事务所。
    泷泽先生是从艺能人的身份隐退的，后来进入公司管理层。

    那泷泽先生也是“姐妹”，有可能会从“妈妈”手中继承茶屋的“姐妹”。
    目黑收到道子的回信，在心里默默地对泷泽说了一句对不起。

 

    目黑君可以不要叫我“道子”吗？
    他的笔迹纤细锋利，那行字写到后面越写越小，歪歪斜斜地横在信纸上。
    我不喜欢这个名字。
    道子是后来取的假名，要让目黑君失望了，我不是女孩子。

    他忐忑地将那封信寄出去。
    没有人喜欢被欺骗，目黑会怎么想他呢。他以为自己会收到愤怒或者是冷落，可是回信里只有一句简短的答复：“好。”
    目黑问他希望自己怎么称呼他。

    可以叫我道枝吗？
    目黑君原本想要找的朋友就叫这个名字吧，请把我当成是他吧。

    道枝告诉他置屋的姐姐正在教他弹奏三味线，在客人面前表演时用的琴是用猫皮和黄金象牙配件钉制的，练习用的则是结实粗糙的狗皮木质琴。
    他觉得自己大概不会有机会碰到那样精美的乐器的了，但是木头做的琴也很好，他已经开始学会演奏长歌。

    每一年庆典结束的时候，梅川茶屋都会安排一台歌舞伎剧。艺伎们在表演这些剧目时，不得不女扮男装出演男性角色，对于看惯了寻常歌舞伎舞台上男扮女装表演的观众，这种倒错的反差无疑十分有趣。
    秋季庆典前夕，梅川突然将那把表演用的三弦琴塞到他的怀里——原本负责出演的女孩子脸上起了疹子，没有办法在客人面前表演了。

    妈妈桑用龟甲梳和珊瑚发卡替他固定好假发，用坚果油和加热的小竹板把它打理得油光水滑。他的脸和脖子上扑了一层粘稠的东西，用来保证白粉能够均匀地附着在他的皮肤上。
    道枝描上红色的眼线和红色的嘴唇，第一次从幕后走到台前。
    尽管不是女孩子，得体的服装和肢体语言让客人们觉得他只是一个个子高挑的普通艺伎。当他坐下的时候，这一点差别便也看不出来了。他小心翼翼地把腿藏进和服里，那些站立时只到他肩膀高的艺伎和舞伎们坐下来以后看上去和他差不多高。

    目黑说他所在的时代也有歌舞伎，在演武场气派的大礼堂里上演，观众席足足有三层。

    我刚开始出演《泷泽歌舞伎》的时候也是一样，原定出演的一位Jr.受伤了，泷泽先生联系我作为他的替角。
    后来加入了新的团体不久，再有机会见到泷泽君的时候，我问他“是因为替角的事情才让我加入的吗？”
    他微笑着说“也许吧”，到现在我都不知道答案。

    道枝有些唏嘘，歌舞伎演员和艺伎曾经被人们认为是声名狼藉的卖艺者，连最潦倒的学生都能够光顾艺伎花街。一个世纪以前还只是大众娱乐的歌舞伎和艺伎竟然在目黑的时代成为了一种高雅的追求，人们仅仅花费金钱尚且不够，还要具备一定的修养才能够欣赏这些艺术。

    他开始在梅川茶屋真正成为一个和女孩子们别无二致的艺伎，却在目黑的信件里第一次以男性的身份扮演一个名叫“道枝”的朋友。

    尽管他并没有和女孩子一样的器官，他还是在衬裙里穿了上浆的棉内衣，又厚又硬的布料贴在皮肤上，闷得几乎透不过气来。
    他趴在木头窗框上看到鸭川河堤下正在漫步的男男女女，那些浪漫的恋人沿着河岸走走停停，女孩子温顺地依偎在男人怀里。他们在夜色的掩护下接吻，像一对交颈的天鹅。

    狼狈的性别认同第一次让他感到无所适从。
    他是一条干涸的金鱼，拼命地往玻璃缸里跳，磕磕绊绊地摔进去以后才发现，原来里面没有水。
    他还是不知道自己是谁，自己是什么。

    “道子，好了没有？”
    他拆下衬裙里的棉内衣，闷闷地在部屋里答应一声。

    宴会厅的壁龛从地板一路延伸到天花板，道枝捧起那把猫皮和黄金打的三弦琴，慢慢地、平稳地在榻榻米上跪坐。
    他将琴弦调到本调子，用象牙拨子拨弄琴弦。
    许多道目光像照明灯一样，一束束地聚集在他的身上。
    干涸的金鱼架在灯上烤。

第2章 幽灵在他身体里变成疯长的的野草

发表于 2022-01-25 | 分类于 大正浪漫

前略

    很抱歉上次的信匆匆寄出忘记落款了，我的名字是目黑莲，如果可以的话请允许我称呼您“道子小姐”。
    道子小姐愿意为我的事情挂心让我很高兴，但是我原本想要送信的对象并非艺伎，应该不会出现在花街附近。

    附信送来的樱花很漂亮，花瓣和雨水的味道留在信纸上，感觉像是嗅到了关西的春天。
    我对上一封信中提到的梅川茶屋很感兴趣，道子小姐是在那里工作的艺伎吗？
    最近我所在的组合团体在关西有一场宣传活动，不知道届时可否叨扰一二？

草草

 

    还没有等到对方的回信，目黑就已经坐在了去往大阪的新干线上。
    电线杆的影子在他的背上飞快地刮过，铁轨和车轮的碰撞盖过了康二的声音。
    “什么？”他凑过去让他再重复一遍。
    “我说，”康二对着他的耳朵，声音很大，“我们去的是大阪，可是先斗町在京都啊。”
    目黑愣了一下，信封上的收件地址写的是关西杰尼斯事务所，邮递公司竟然不小心把它送到京都去了。

    “先斗町是什么样的地方？”
    列车钻进隧道，白色的照明灯连成线地飞奔而来。
    “听说以前是有名的花街。”
    先斗町和祇园、上七轩同属京都六大艺伎区，外事活动中那些在摄像机前为各个国家的政府首脑斟日本米酒的艺伎大多出身于这几条花街。
    “这种事情我也不太靠谱啦，你怎么不在雅虎上搜搜？”
    康二耸一耸肩，他是大阪府出身，小时候还在泰国待了几年，对京都的花街着实不熟。

    目黑在搜索栏里输入“梅川茶屋”，只跳出了几条不相干的网址链接。
    “说不定只是没有来得及更新信息而已，”康二拍拍他的肩膀安慰道，“回头我们一起去找找。”

    活动结束以后他们从心斋桥坐地下铁到新大阪站，再转乘JR在京都下车。
    先斗町挤满了前来赏樱的观光客，密密麻麻的人头攒动，大大小小的居酒屋和酒吧林立，带着古建筑的风情。
    他们钻进一家又一家挂着白色招牌的店面，从街头走到街尾——没能找到梅川茶屋。

 

前略

    称呼的事请随意。
    先斗町每隔半年都会举办一次为期三周的庆典活动，艺伎们会在歌舞练场表演鸭川踊和浪花踊。虽然我并不能被称为是“艺伎”，也不擅长演奏三味线，但是如果目黑君不嫌弃的话我会跳浪花踊。
    我的房间正对着鸭川河的西岸，一周以后的周六晚上六点半（茶屋六点半开始营业）我们在鸭川河见。

    另，请问目黑君前一封信提到的“事务所”是什么？还有LINE和SNS的意思我也不太明白。
    不过我想这些都不重要啦，我们可以当面再说，如果目黑君愿意的话。

梅川茶屋
道子

 

    目黑收到回信以后，在下一个周六又专程去了一次京都，康二也跟着一起来了。
    “你放心，”康二举起右手竖着三根手指，“只要道子小姐一出现我就立刻消失，绝对不给你当电灯泡。”
    目黑无力地解释：“我和她只是通了几封信的关系。”

    先斗町位于鸭川河西岸，东西走向的三条大街和四条大街划出了它的南北边界，这两条街横跨鸭川河，分别形成三条桥和四条桥。
    一整条街的樱花落在水面上，鸭川河流淌成粉红色，两岸的木质房屋搭建起宽广的游廊，长长的廊道一直延伸到河岸上，檐下挂起一排纸灯笼。
    他们在桥梁和游廊之间穿梭，从六点半一直等到九点。
    天空缓缓睡去，疲倦的暮色舒展铺开又收敛消散。目黑等了很久很久，对着鸭川河西岸每一扇打开的窗子大喊“道子小姐”的名字。
    没有得到任何回应。

    “别喊啦。”
    他回过头来，看到一个皱巴巴的、头顶带有一块圆形秃斑的老人。
    如果你在日本看到某个年长的女性头顶有这样一块圆形秃斑，那么她十有八九在年轻的时候做过舞伎——舞伎的基本发式需要揪紧头顶的一小缕头发，时间久了以后这缕头发就会渐渐掉光不再生长。
    老人骄傲地昂起脑袋，没有用头发把那片裸露的头皮遮住。这块秃斑记录了她过去承受的艰苦，被她视为荣耀的象征。
    “梅川茶屋？”她拉长声音，慢吞吞地道，“那是很多年以前的事了，快要有一百年了吧，早就没有了。”
    目黑又向她问起道子。
    老人嗤了一声：“没听说过。”

    鸭川河在月光下像一条绸带，目黑静静地看着，突然想起道子在信里问他，LINE是什么。

 

    道子踏上系有铃铛的高齿木屐在鸭川河岸等他，和服是精心挑选的，白色的柳枝刺绣从腰臀的位置开始像瀑布一样垂下，金色的浪花在留袖的边缘翻涌。
    艺伎和服上的纹样会根据季节变换主题，时间和鸭川河一起不紧不慢地流淌，他换下五月的袷（带衬里的和服），穿起六月的絽（纱罗和服），春天的梅花变成夏天的菖蒲，目黑不仅没有赴约，而且还迟迟没有给他回信。

    夏天结束的时候经纪人给他发了邮件，一个月以后Jr.组合浪花男子正式结成，他站在松竹座的舞台上，终于看到了目黑看过的风景。
    庆功宴上他和队友一起拍了合照，第一次觉得自己有资格给目黑发消息了。目黑的旧手机像素不太好，拍出来的照片模模糊糊的，演出服上带着流苏和亮片，他在镜头前笑得有点傻。

    目黑没有回复他。
 
    “怎么了道枝，你不是一直说想来六本木看彩灯吗？”
    大桥和长尾远远地把他落在后面，高桥提着满手购物袋停下来等他。
    蓝色和黄色的节日彩灯缠绕在树上，商业街放着圣诞颂歌，他漫无目的地路过透明的玻璃橱窗和飘着麦子香味的面包房，灯光在他的脸上迁徙。
    “你没有什么想买的吗？”
    道枝看中了一件格子外套，袖子长长的中性款。他在Snow Man冠番的综艺里看到过，目黑选给恋人的也是这样一件黑白的格子外套。
    他在试衣间的镜子面前站了很久，还是把它脱了下来。
    钱不太够，下次再来买吧。

    夜风冷得像磨砂纸，他呼出白色的气团，揉搓着冻得红红的手，有那么一瞬间感觉自己好像回到了那个冬天，他趿着目黑的拖鞋，在洪水猛兽一样的陌生城市里毫无逻辑地走。
    一只巨大的轻松熊截住了他的脚步，他牵着一大束气球，灯光穿过彩气球像在调色板上滚了一圈似的，五光十色地画在他的脸上。
    气球的绳子缠在一起，他从上面拆下来一颗黄色的递给道枝。
    “给我的？”
    道枝愣了一下，慢慢地把通红的手指从袖子里露出来。
    轻松熊点了点头，然后招手示意他凑过来，让他把手伸进自己腹部的大口袋里。
    一大束气球浮在他们的头顶，像一棵枝繁叶茂的树，彩色的树冠下，道枝摸到轻松熊软绵绵的白肚皮，手指尖碰到了什么东西。
    他把它拿出来，发现是一双厚厚的羊毛手套。
    “这个也是给我的吗？”
    他眨着漂亮的眼睛，柔软的发尾扫在衣领上，像猫咪、兔子、雨水淋湿的小狗、翻滚在地的幼狮，像所有小动物。
    轻松熊隔着毛茸茸的布偶服，轻轻地碰了碰他的脑袋。
    “道枝，好了没有？”高桥的声音远远地传来。
    他如梦初醒地匆匆道谢，牵着那颗黄气球不一会儿就跑远了。
    天空飘起细雪，目黑摘下沉甸甸的头套甩了甩头发，糖霜似的雪融化在他的头顶。
    他走进店里，把道枝试过的那件黑白格子外套买了下来。

    回到酒店以后道枝把黄气球戳破，放了气以后展平塞进旅行箱里。
    高桥在穿衣镜前试演出服，31号的跨年演唱会道枝没有办法登台，未成年人工作有非常严格的宵禁制度，他只能在观众席看着。摄像机扫到他的时候在他身上停留了六秒，这就是他在整场演唱会中的全部镜头了。
    演出结束以后他捧着一个盒子在后台寻找目黑的身影，Snow Man的白色衣服在后台的昏暗光线下非常显眼，没过多久他就看到了行色匆匆的佐久间。
    康二和岩本一左一右地架起目黑的肩膀，佐久间冲上去：“目黑怎么烧成这样了？”
    “从圣诞节开始就一直低烧了，昨天彩排回来体温一直下不来，也不知道他那天到底干什么去了。”
    目黑垂着脑袋处在清醒和昏迷的边缘，整个人像刚从水里捞出来似的。
    “喂喂，你没事吧，”佐久间用纸巾擦他头上的汗，“小道枝应该也在这里，要不我把他给你找来？”
    “不要……”他的声音被重感冒撕成碎片。
    “什么，”佐久间没有听清楚，“不要道枝？”

    道枝捧着那个小小的盒子胶在原地。
    盒子还在手里，心却掉在地下。
