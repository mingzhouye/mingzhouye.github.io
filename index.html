第13章 三千惠

发表于 2022-02-03 | 分类于 吉原炎上

    “房间里只有您一个人。”
    目黑的呼吸停住了，那一瞬间他的脑海里浮浮沉沉地闪过许多冗杂的片段，他在混乱中拼命地找寻，直到那些与道枝相关的记忆被海水冲刷上岸。
    为什么第一次见面的时候，游女屋的伙夫赶走了所有的新造和秃却偏偏漏下了道枝？
    为什么引手茶屋的吉原细见上没有道枝的画像和名字？
    为什么当他告诉道枝，自己在夜见世看到他了的时候他表现得那样不可思议？
    为什么卖章鱼烧的小贩会反复和他确认是不是要两人份的食物？

    为什么道枝一次又一次地拒绝了他，又偏偏不告诉他为什么自己不能离开千代屋。
    那些记忆涨潮一样拍打向他，淹过脚脖子，淹过膝盖，一口气地奔涌上来，没过他的头顶。

    “又来了，那个奇怪的客人。”
    “你说谁啊？”
    “一层最里面那间客房的常客，姓目黑，听说以前是个武士，后来腿脚不利索了。”
    “目黑……你是说‘目黑不动’的‘目黑’？”
    “为什么这么说呀？椿姐姐说那位大人是个挺温柔的人啊！”
    “他太奇怪了，别人来吉原都是找乐子来的，扬屋里热热闹闹的，隔着走廊都能听见里面的声音。只有他每次来既不指名游女也不叫芸者，自己一个人推着轮椅进去，有的时候一坐就是一整宿。”
    “老板娘看他一个人冷冷清清的，让遣手嬷嬷带女孩子进去陪他，他反倒还生气了，沉着脸把人往外赶。”
    “怎么会有这样的人，真奇怪！”
    “嘘，小点声，别叫客人听见了……”

    凑在走廊里叽叽喳喳说着闲话的新造和秃们被伙夫轰走，椿把席上的芸者和帮间都请出扬屋，提出要单独和目黑说句话。
    遣手嬷嬷在大开的两面隔扇门外，吞吞吐吐地提醒她：“花魁，这不合规矩吧，初会的时候怎么可以与客人单独相处……”
    椿顶着满头发簪和笄子，高高的、立兵库式样的发髻和上面点缀着的沉甸甸的珊瑚、珍珠和翡翠给了她底气，她微微扬起下巴，和目黑在长谷川房间里看到的那个瑟瑟发抖的小女孩判若两人。
    年轻的花魁加重了语气：“我说了，请出去。”
    芸者将三味线收起来装进琴箱里，帮间把手里的折扇一合，也跟着正要离开。
    “请等一等。”目黑叫住了他，“请把那个故事讲完。”
    “您是说刚才的故事吗？”帮间停下脚步，“刚才席上的那个故事已经讲完了。”
    “不，是祭典上那个。”
    目黑定定地与他对上视线，前所未有的恐慌，又前所未有的镇静。
    为什么道枝会仅仅因为昼见世迟到了一会儿就被遣手用藤鞭抽打得皮开肉绽？
    因为他消失不见了，游女屋以为他要逃跑。
    名叫三千惠的游女，玳瑁框的西洋镜，玻璃弹珠、雏人偶、万华镜……
    他怎么可以这样迟钝。

    落语师在蒲团上重新落座，娓娓地讲述起祭典上被烟花打断的余下半截故事。
    “三千惠因为一副西洋镜被打得全身上下不剩一块完好的皮肉，她疼得昏迷过去，又被人用冰冷的井水浇醒，就这样煎熬了三天三夜也不肯承认自己偷了东西，反倒是编出了个故事来糊弄人。”
    “她说她的妈妈是目黑不动寺庙里的巫女，怀孕以后逃到关西的一座神社里生下了她，可惜没过几年就死了，只给她留下一件振袖、一把桧扇和一柄神乐铃。”
    目黑的眼前浮现起男孩用白檀纸绑起发尾，持着铃铛和桧扇在庭院里起舞的样子，他披着雪白的振袖，像是把月光穿在身上。
    落语师放缓了语速，手中的折扇“唰”地一下打开又合上。

    “奄奄一息的游女告诉千代屋的主人，侍奉五色不动的巫女是神的孩子，有穿越时空的能力。”

第12章 花魁

发表于 2022-02-03 | 分类于 吉原炎上

    “是的，我要为他赎身。”
    盂兰盆节祭典后的整整一个月，目黑都没有再见到道枝。
    庆应四年七月，睦仁天皇颁布诏书成立了管辖江户町奉行支配区域的东京府，江户不再被称之为江户，吉原却还是原来的吉原。
    和室里的隔扇门似乎成了摆设，道枝已经很久没有推开过那扇门了，目黑犹豫再三，还是再一次找上了千代屋。
    “您是说朝雾花魁房间里的振袖新造？”老板娘把长烟管搭在烟草盆上，轻描淡写地开口，“朝雾得淋病死了，她房间里的振袖新造是新的花魁。”
    “您要为她赎身得按花魁的规矩来，这恐怕要花掉您相当一笔钱。”
    目黑毫不犹豫：“我不在乎。”

    花魁的新客想要变成熟客必须经过一系列繁琐的步骤，而想要为她赎身，成为熟客是必须的步骤。
    游女屋的客人与花魁的第一次见面被称之为“初会”，这个时候别说是行房了，花魁初次见生客连一句话都不会和客人交谈，如果认为对方的品性不适合自己还可以将他拒之门外，被拒绝的客人从此再也无法踏进同一家店的店门。
    初会过后的第二次见面叫做“回返”，这时花魁对客人的态度仍然冷淡，直到第三次见面双方彼此相熟，花魁才会与客人同床共枕。
    目黑通过引手茶屋给妓馆送了指名信。幕藩制度废除以后武士不再高人一等，门阀制度也在一点一点瓦解，但目黑还是穿上了最正式的、绘有家纹的御纳户色纹付，按照约定的时间来到了千代屋。
    扬屋里的伙夫替他安置好拐杖，目黑在壁龛的正前方落座。从地板一只延伸到天花板的壁龛中挂着一副鲜艳的风俗画，挂画下是插在瓷瓶里的高砂百合和紫色桔梗，芸者弹奏着三味线哼起长歌，助兴的帮间活跃着气氛讲起俏皮话。那个帮间目黑见过，正是他和道枝在祭典上看到的那个胖胖的、脸颊像白面馒头一样的落语师。
    没过多久就有侍者送上茶食，开胃的醋腌菜和味增汤后是装在漆碗和黑陶盘里的小菜，金枪鱼、鲣鱼、鲑鱼、海胆，肉汤里红色的鲷鱼头眼珠子仿佛瞪着人，最后一盘是腌渍过的甜海鳗芋头——他第一次见到道枝的时候，男孩就缩在隔扇门边上，店里的伙夫赶走了跪坐在走廊里吃客人剩菜的新造和秃，只有道枝还在原地，猫儿一样叼着一块甜口的狼牙鳗。
    主菜上桌以后花魁才姗姗来迟，迟到也是流程的一部分。两个看上去约摸七八岁的秃一左一右地拉开隔扇门，遣手在走廊里出声提醒客人，花魁到了。
    第一次正式见客的花魁穿着厚重的祢裆，头上戴着三把附有金箔的柘植拢子、三对黄杨木质地的蒔绘发簪，宽大的腰带系在胸前，像抱着一只五颜六色的小枕头。
    目黑的视线上移、定格，接着趔趄着起身，几乎要打翻面前呈放着精致菜肴的小桌。
    “姑娘你是不是走错了，我要找的人是道枝。”
    按照吉原的规矩，初会时客人是不允许主动向花魁搭话的。三味线的声音停了下来，等候在廊道里的伙夫扒着和室门，随时准备听里间一声令下动手赶人。
    椿睁大眼睛，困惑地眨了眨：“道枝是谁？”
    “朝雾花魁房间里的振袖新造，”目黑急切地补充道，语速飞快，“是个男孩子。”
    “大人，您说笑了，千代屋是游女屋，怎么会有男孩子？”
    又是这句话，他第一次走进引手茶屋的时候，茶屋里的伙计也这样对他说。
    目黑觉得自己的喉咙像是被堵住了，他吞咽了一口唾沫，声音艰涩得像从管道里挤出来的一样：“他在这里，一层走廊最里面的房间，我每一次就坐在那里等他”
    “你也看到他了吧，”目黑死死地盯住椿的眼睛，好像在抓住最后的救命稻草，“长谷川的那次，你还记得他吧？”
    椿大可以不理睬他，甚至可以叫外面的帮间把眼前这个不知所云的客人赶走。但她还是得体地笑了笑，耐心地对他解释：“是的，大人，我记得很清楚。”
    “房间里只有您一个人。”

第11章 落语

发表于 2022-02-03 | 分类于 吉原炎上

    目黑把章鱼烧塞给他以后，又一瘸一簸地去给他买刨冰。
    “去那边坐下等我。”
    道枝认命地捧着两人份的章鱼烧，到底是谁腿脚不方便啊，他不放心地目送着他的背影，在心里暗暗地想。
    他顺着目黑指的方向看过去，转角的地方有一处临时搭建的寄席，一个脸颊像白面馒头一样的、胖胖的落语师正坐在一张软垫子上，绘声绘色地讲起落语故事。
    所谓“寄席”就是集中表演落语的小剧场，坐下来歇脚的行人围坐在落语师的周身听得津津有味，时不时被逗得前俯后仰。
    推着刨冰车的小贩把刨成雪花状的冰渣倒进一支用纸卷成的锥形筒中，细细的冰沙顶上淋了红色的糖浆，不知道是梅子味的还是甜李子味的，目黑拿着它好不容易才从人群中挤出来，磕磕绊绊地回到道枝身边。
    章鱼烧已经晾得不再烫口了，目黑一口一个，三两口就吃掉了一整份，当他放下竹签的时候道枝还在含着木头勺子慢慢地吮，纸筒里的刨冰连淋着糖浆的顶都没凹下去。
    “刚刚他前面讲了什么？”
    他去买刨冰的时候耽误了一阵子，错过了半截故事。
    “是游女的故事。”道枝附在他的耳边，一呼一吸带着刨冰的温度，“有个叫三千惠的游女突然不见了。”
    “她消失得没有太久，昼见世的时候就回来了。起先遣手嬷嬷以为她只是和其他女孩子一样，偷偷地跑去出会茶屋见情郎了，于是便睁一只眼闭一只眼没有管她。”
    “可是后来她不见的次数越来越多，有一次甚至一整夜都没有回来，连昼见世都没有赶上。游女屋以为她在策划出逃，老板娘带着店里最得力的伙夫，几乎把整个吉原都掀翻了一样到处找她，结果发现她其实就在庭院里，哪里也没有去。”
    落语师将折扇一合，“啪”地在手心里一打，衔接上了道枝讲给他的前情提要。
    “谁料到这第二天，扬屋里就有客人丢了东西。”
    “丢东西的是位年近六旬的老人家，一进扬屋就大喊着‘眼镜，我的眼镜呢’，”他眯起眼睛，学着老人的姿态佝偻着脊背，夸张的动作逗得座下的客人又是一阵哈哈大笑。
    目黑发现道枝没有笑，他连纸筒里流出来的刨冰汤化在手心里都没有意识到。目黑从怀里抽出手帕替他擦干净手。
    “于是遣手们就在妓馆里找，她们打开每一台柜子，划破每一只枕头，终于在一块榻榻米的下面找到了一副玳瑁框的西洋镜——正是三千惠的房间。”
    “遣手还从她的房间里搜出了好些新奇的小玩意儿，小孩子玩的玻璃弹珠、五颜六色的万华镜，还有各种各样的雏人偶。三千惠偷了这么多东西，却偏偏辩解说自己没有，气得那个丢了西洋镜的客人连连咳嗽，指着她的鼻子痛骂。”
    “再后来啊……”
 
    他们没有听到故事的后续，道枝听到某种空心的爆破声，人群一下子骚动起来，坐在软垫子上听落语的人们一下子全都站起来，落语师便闭嘴不再讲了。
    茂密的烟花争先恐后地将天空填满，照亮了每一张期盼的脸，小孩子们坐在父亲的肩上，朝着头顶的方向兴奋地挥手。散开的烟花像流动的火，一朵比一朵盛大，一朵比一朵热烈。金色的雨在半空中消失不见，像断线的珠子，像枯萎的花瓣，很快就有下一束花火将它取代。
    道枝怔怔地看着远方的天幕，嘴里含着最后半个已经凉掉了的章鱼烧。
    目黑攥住他的手腕，又短暂地松开，重新扣住他的手指，看上去似乎有话要对他说。
    他赶在目黑之前开口：“我要回去了。”

第10章 祭典

发表于 2022-02-01 | 分类于 吉原炎上

    目黑不止一次提出过想要带他离开吉原。
    “你看，我现在可以站起来了，”他把拐杖靠在墙上，展开双臂试图保持平衡，“莫雷尔先生说我恢复的状况很好，很快就能像正常人一样走路了。”
    道枝的睫毛剧烈地抖动了一下，不是因为这个。
    他看到目黑坐在轮椅上的时候，总是不受控制地想到仲之町的纸艺匠人手中插在竹筒里的剪纸蝴蝶，残缺的部分和保留的部分都是它的翅膀。
    当你看到一轮弯月挂在天上的时候，会认为它是有缺陷的吗？
    在道枝的眼里，圆月和弦月都是装裱在窗框里的风景画。

    目黑不再追问他为什么不肯离开，只是到千代屋来的频率越来越高。
    仲之町会自七月十五日的盂兰盆节开始举办持续两个星期的祭典，夜间小摊挨挨挤挤地连成片，戏剧班子和江湖艺人使得整个游廓都无比热闹。
    “这个是……给我的？”
    目黑带给他一套藏青色的细条纹和服，麻质的小袖外面是深色的羽织，但这些都不是重点。道枝微微睁大眼睛，这是一套男式和服。
    他在游女屋生活了将近十年，像女孩子一样挽起头发，画细细的眉毛，簪柘植拢子和细绢花，没有人把他当男孩子看。
    目黑不好意思地朝他笑笑：“是我以前穿过的，现在做新的来不及了。”
    来不及了？
    没有时间多想，他换上目黑的旧衣服，套上足袋，抚平身上的褶皱以后站直身体。目黑把他的头发拢在手心扎成一个短短的发辫，牵着他的手走在仲之町的街道上。
    红色的纸灯笼点亮了一整条街市，他们轻而易举地就被汹涌的人潮淹没。目黑的下肢还没有完全康复，他走得很慢，一只手拄着拐杖，慢慢地挪动步子，另一只手紧紧地拉住道枝的手腕。
    道枝落后半步跟在他身后，像一个寻常人家出身的、第一次来赶祭典的普通男孩一样，好奇地四处张望。走过拐角的时候他放缓步伐，在一面红栅栏前驻足。游女在游廓之内有一定范围内的自由，她们可以在白天捧着装有米糠袋的木盆结伴去浴汤洗澡，也可以去菓子屋买酥脆软糯的金锷烧与艾蒸糕，甚至独自一人偷偷前往出会茶屋与情夫幽会，但是到了张见世的时间必须回到笼子里。
    他顺着木质的红格窗朝里面看进去——目黑君那个时候也是这样看到他的吗？
    红栅栏里的游女伸着手臂递出一支长烟管，薄薄的烟雾像一层帘子一样将他与里面那个光怪陆离的世界分割，他皱起鼻子打了一个小小的喷嚏，目黑把五指插进他的指缝里，用一种很有侵略性的势头扣住他的手指，把他拉走了。
    他们路过沙沙作响的刨冰摊和挂满狐狸面具的假面屋，捞金鱼的摊位前红色和白色的金鱼在水缸里摆动尾巴，小孩子们吵吵嚷嚷地你追我赶，章鱼烧留下的汤汁还留在嘴边没有擦干净。
    道枝轻轻地“啊”了一声，目光停在某个方向定住了。目黑顺着他的视线看过去，卖章鱼烧的小贩正把上半身探出摊位，用带着一点关西腔的口音吆喝着。
    “我的家乡也有这个，在我还很小很小的时候，妈妈会买给我。”
    目黑当即拉着他在摊位前停下：“要两份章鱼烧，谢谢。”
    “两人份？您确定吗？”周围的环境太吵了，小贩似乎没有听清楚，回问了一句。
    目黑点一点头，伸手去接食物的时候放开了他的手。
